# BitoFamily

We build educational apps that make learning feel like a game — and make progress feel real.

Bito is a universe of subject-based apps for children in Grades 1–3, each featuring the same cast of robot companions: **Coco**, **Kato**, and **Sona**. Every star a child earns inside a Bito app is redeemable for a real-world reward chosen by their parent — turning screen time into a motivation engine, not a distraction.

## Platform

| Repo | Role |
|---|---|
| [BitoCore](https://github.com/BitoFamily/BitoCore) | Shared character assets, UI themes, and common game logic used across all Bito apps |
| [BitoBackend](https://github.com/BitoFamily/BitoBackend) | Cloud sync, user authentication, and cross-app data layer |
| [BitoLocalization](https://github.com/BitoFamily/BitoLocalization) | Translation pipeline — English, French, Hindi, Arabic |
| [BitoParents](https://github.com/BitoFamily/BitoParents) | Parent companion portal for managing rewards and tracking progress |
| [BitoWeb](https://github.com/BitoFamily/BitoWeb) | Public website and marketing |

## The characters

**Coco** (Puppy-Bot) · **Kato** (Boy-Bot) · **Sona** (Girl-Bot)

Each companion has a distinct personality and color palette, and guides children through their subject's challenges with encouragement, celebration, and a bit of friendly competition.
